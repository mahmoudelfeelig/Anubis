# LV-015 - Graph Braid Loom

**Username**

The dependency chain frees exactly one spool at a time:

1. Glyph Node  
2. Ravel Span  
3. Anchor Gate  
4. Pylon Brace  
5. Helix Courier  
6. Brace Array  
7. Relay Strand  
8. Aerie Pivot  
9. Inlet Cradle  
10. Drift Verge

Taking the initials yields `graphbraid`.

**Password**

For each row, sum the pair in each warp and reduce modulo 2 to form a five-bit word. Converting each binary word (most significant bit at Warp α) with `0 → a` produces:

1. `01111` → `p`
2. `00000` → `a`
3. `10001` → `r`
4. `01000` → `i`
5. `10011` → `t`
6. `11000` → `y`
7. `01011` → `l`
8. `01110` → `o`
9. `01110` → `o`
10. `01100` → `m`

Thus the password is `parityloom`.
