# LV-012 - Eigen Lattice

1. Compute \(A^2 + A\) for the given matrix. Squaring and adding produce
   \[
   M = A^2 + A = \begin{bmatrix}6 & 6 & 1\\ 1 & 12 & 6\\ 5 & 1 & 6\end{bmatrix}.
   \]
2. Solve \(Mx = b\) for each column in \(B_u\). Using Gaussian elimination:
   - \(b_1 = [91, 155, 76]^T \Rightarrow x_1 = [5, 9, 7]^T\).
   - \(b_2 = [121, 215, 150]^T \Rightarrow x_2 = [5, 14, 7]^T\).
   - \(b_3 = [166, 150, 123]^T \Rightarrow x_3 = [18, 9, 4]^T\).
   Reading the results row-wise yields numbers \([5, 9, 7, 5, 14, 7, 18, 9, 4]\), mapping to `eigengrid`.
3. Solve \(Mx = b\) for each password column in \(B_p\):
   - \(b_4 = [202, 222, 129]^T \Rightarrow y_1 = [18, 15, 4]^T\).
   - \(b_5 = [143, 332, 131]^T \Rightarrow y_2 = [2, 19, 17]^T\).
   - \(b_6 = [79, 210, 89]^T \Rightarrow y_3 = [0, 11, 13]^T\).
   - \(b_7 = [198, 290, 159]^T \Rightarrow y_4 = [14, 17, 12]^T\).
   Flattening these numbers gives \([18, 15, 4, 2, 19, 17, 0, 11, 13, 14, 17, 12]\), which spells `spectralnorm` under the 0→`a` mapping.
