# LV-011 - Chronometric Vector

1. Convert each local tick into UTC using the stated offsets. The nine UTC times are:
   - Chronicle Post → 02:02
   - Helix Relay → 05:07
   - Relay Fold → 15:17
   - Obsidian Gate → 17:14
   - Nimbus Array → 19:13
   - Oracle Spine → 21:14
   - Kepler Verge → 22:10
   - Echo Stack → 23:04
   - Yonder Node → 23:24
2. Sort the lines by these normalized times and read the station initials in that order to get `chronokey`.
3. Take each UTC offset (in hours), add 12 to recenter the vector on zero, and map 0→`a`. The shifted values are `[20, 19, 2, 21, 4, 2, 19, 14, 17]`, which spell `utcvector`.
